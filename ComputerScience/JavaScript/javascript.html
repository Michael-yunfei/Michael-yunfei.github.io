Development Tools

Unlike Python, Javascript does not have its own interpreter environment at the very beginning. You need run your code through a browser. It could be frustrated for beginners to learn this language without knowing HTML and CSS. So, where should we start to learn Javascript? Here are some choices:

  1. PlayCode: you need pay to access the full serivce.
  2. CodePen: it's free and quite popular, but the layout is not as beautiful as PlayCode.
  3. WebStorm: use this one once you feel comfortable with JavaScript.

Run JavaScript with Atom

If you don't want to buy any service from any of above websites. I still highly recommend use Atom with script.
