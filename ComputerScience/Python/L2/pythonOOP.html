What is object-oriented programming?

A Pythonic Card Deck

The following is a very simple example, but it demonstrates the power of implementing just two special methods, __getitem__ and __len__. 

import collections

Card = collections.namedtuple('Card', ['rank', 'suit'])


class FrenchDeck(object):
    ranks = [str(n) for n in range(2, 11)] + list("JQKA")
    suits = 'spades diamonds clubs hearts'.split()

    def __init__(self):
        self._cards = [Card(rank, suit) for suit in self.suits for rank in self.ranks]

    def __len__(self):
        return len(self._cards)

    def __getitem__(self, position):
        return self._cards[position]


deck = FrenchDeck()
len(deck)
deck[3]

We have just seen two advantages of using special methods to leverage the Python data model:

  • The users of your classes don't have to memorize arbitrary method names for standard operations.
  • It's easier to benefit from the rich Python standard library and avoid reinventing the wheel.
